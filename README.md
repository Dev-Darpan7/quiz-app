# 🧠 Quiz App

A simple interactive Quiz App built using **HTML, CSS, and JavaScript** to practice DOM manipulation, event handling, arrays, objects, functions, and basic responsive design.

## 🚀 Features

* 📋 Multiple-choice questions
* ✅ Instant correct/wrong feedback
* ⏳ Automatic delay before the next question
* 🔒 Prevents multiple answers for the same question
* 📊 Displays current question number
* 🏆 Final score at the end of the quiz
* 🔄 Restart Quiz button
* 📱 Responsive layout for smaller screens

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 📚 Concepts Practiced

* Variables
* Arrays & Objects
* Functions
* DOM Manipulation
* Event Handling
* Conditional Statements (`if/else`)
* Loops (`forEach`)
* `querySelectorAll()`
* `setTimeout()`
* Button disabling using the `disabled` property
* Responsive Design with CSS

## 🎯 Future Improvements

* Add more questions
* Shuffle questions and answer options
* Add a timer for each question
* Show the correct answer after a wrong attempt

## 💻 What I Learned

While building this project, I learned how to:

* Update the webpage using DOM manipulation.
